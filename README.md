# Placas
Projeto demo de e-commerce desenvolvido para simular uma loja virtual de produtos personalizados, como placas, pins, broches, crachás e chaveiros. O site apresenta catálogo de produtos, seções institucionais, clientes, integração com WhatsApp e layout responsivo. Desenvolvido com HTML5 e CSS3 para fins de estudo, prática e composição de portfólio
